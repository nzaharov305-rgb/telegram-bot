# Деплой бота на Railway

## Шаг 1. Создать проект на Railway

1. Зайдите на [railway.app](https://railway.app)
2. Войдите через GitHub
3. Нажмите **New Project**
4. Выберите **Deploy from GitHub repo**
5. Подключите репозиторий с ботом (или **Configure GitHub App** для доступа к репо)

> Если репо нет — сначала загрузите код на GitHub:
> - Создайте репо на github.com
> - `git init` в папке проекта
> - `git add .` и `git commit -m "init"`
> - `git remote add origin https://github.com/ваш_логин/имя_репо.git`
> - `git push -u origin main`

---

## Шаг 2. Добавить переменные окружения

В Railway откройте ваш сервис → вкладка **Variables** → **Add Variable**:

| Переменная | Значение |
|------------|----------|
| `TOKEN` | Токен от @BotFather |
| `ADMIN_ID` | Ваш Telegram ID |
| `PAYMENT_CARD` | Номер карты для оплаты |
| `PAYMENT_AMOUNT` | 2990 |

## Шаг 3. Volume для базы данных (важно)

**Зачем:** Без Volume при каждом перезапуске бота база (пользователи, платежи) обнуляется.

**Где это в Railway:**

```
railway.app → Ваш проект → Клик по сервису (бот) → Слева меню
```

### Добавить Volume — по шагам

1. Зайдите на **railway.app** и откройте свой проект.
2. Нажмите на **карточку сервиса** (там, где деплоится бот).
3. Слева откроется меню. Найдите пункт **Volume** или кнопку **+ New**.
4. Нажмите **+ New** → выберите **Volume**.
5. Появится форма:
   - **Mount Path** — впишите: `/data`
   - Нажмите **Add**.
6. Volume создан.

### Добавить переменную DATABASE_PATH

1. В том же сервисе слева выберите **Variables**.
2. Нажмите **+ New Variable** или **Add Variable**.
3. Введите:
   - **Name:** `DATABASE_PATH`
   - **Value:** `/data/database.db`
4. Сохраните (обычно сохраняется автоматически).

---

## Шаг 4. Деплой

1. Railway сам соберёт проект при push в GitHub
2. Или нажмите **Deploy** вручную
3. В логах должно появиться: `Бот запущен`

---

## Шаг 5. Проверка

Откройте бота в Telegram и отправьте `/start`.
